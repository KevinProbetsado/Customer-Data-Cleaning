# Customer Call List Data Cleaning

Cleaned a raw customer call list using Python and pandas.

## What it does
- Removes duplicates
- Drops useless columns
- Cleans last names and phone numbers
- Formats phone numbers as XXX-XXX-XXXX
- Removes customers who opted out (Do Not Contact)
- Splits address into Street, State, Zip Code

## Tools Used
- Python
- pandas

## How to Use
Add your own `Customer Call List.xlsx` with these columns:
`Last_Name, Phone_Number, Address, Do_Not_Contact, Paying Customer, Not_Useful_Column`
